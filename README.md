# Recomendador Personalizado de Conteúdos Educacionais

Este repositório apresenta uma versão inicial de demonstração pública do projeto Personalized Content Recommender.

A proposta é explorar como dados de interação com recursos digitais podem apoiar a personalização da aprendizagem em ambientes virtuais de ensino e aprendizagem.

## Objetivo

Demonstrar uma solução experimental para acompanhar o consumo de conteúdos educacionais e apoiar recomendações personalizadas com base em dados de engajamento, participação e desempenho.

## Publicação com GitHub Pages

Para publicar:

1. Acesse Settings no repositório.
2. Entre em Pages.
3. Em Build and deployment, selecione Deploy from a branch.
4. Escolha a branch main e a pasta /root.
5. Salve a configuração.

A URL pública deve seguir este formato:

```text
https://personalized-content-recommender.github.io/demo-repository/
```
